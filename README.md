# m1-dataviz_reseaux-ferroviaire

Projet de M1 MIASHS en data visualisation et storytelling, visant à explorer un jeu de données au choix, en extraire une analyse et produire une présentation.

Le jeu de données provient d’ici : https://github.com/malin84/transportation_networks_of_china

Il a été exploré avec des packages Python (Pandas, etc.), et les visualisations ont été réalisées avec Matplotlib. (Après plusieurs essais avec Folium qui n’ont pas abouti en raison d’erreurs dans le jeu de données.)
